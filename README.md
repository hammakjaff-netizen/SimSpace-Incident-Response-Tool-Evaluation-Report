# SimSpace Incident Response Tool Evaluation Report

This repository contains a professional technical assessment report based on a controlled evaluation of incident response and security analysis tools in the SimSpace Virtual Range.

## Project Summary
The assessment reviewed four tools used across Windows and Kali Linux virtual machines to support different parts of an incident response workflow:

- **Wireshark** for network traffic monitoring and packet analysis
- **IDA Pro** for malware analysis and reverse engineering
- **Nikto** for host and web server analysis
- **John the Ripper** for password auditing and credential security review

The goal was to evaluate how each tool supports detection, investigation, validation, and remediation planning in a simulated security environment.

## Key Findings
- Wireshark provided strong visibility into packet activity, protocol behavior, and suspicious communications.
- IDA Pro supported DLL and binary analysis for malware investigation and code-level review.
- Nikto helped identify web server weaknesses, outdated components, and configuration problems.
- John the Ripper supported password auditing workflows and highlighted credential security risks.
- The assessment showed that incident response is more effective when multiple tools are used together rather than relying on a single platform.
